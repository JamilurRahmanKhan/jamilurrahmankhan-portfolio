Thesis Project

ML-Based Performance Prediction and Optimization of FSO Links

ML + Communication Systems

This thesis project analyzes the performance of Free Space Optical (FSO) communication links using physics-based modeling, machine learning, and real environmental data. The system predicts key link metrics such as SNR and BER under varying distance and weather conditions, and visualizes the degradation caused by rain through an interactive comparison interface. A Random Forest Regression model is used to predict BER, while theoretical FSO link equations are used to simulate baseline system behavior and support optimization analysis.

Python Machine Learning Random Forest Regression FSO Communication SNR / BER Analysis Data Visualization Weather-aware Modeling Simulation
